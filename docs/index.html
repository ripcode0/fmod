About

This repository provides a pre‑built API based on FMOD. It includes precompiled binaries and headers for seamless integration without building FMOD from source.

  • Supports FMOD Core API v1.10.20
  • Provides pre‑built DLLs (Windows)
  • CMake integration for painless setup

Usage

Standalone Build

git clone https://github.com/ripcode0/fmod.git
cmake -S . -B build -DENABLE_EXAMPLE=ON
cmake --build build

Install

cmake --install build --prefix <install path>

Root CMakeLists.txt

set(CMAKE_PREFIX_PATH <install path>)
find_package(fmod REQUIRED)

# Link to your executable
add_executable(myapp main.cpp)
target_link_libraries(myapp PRIVATE fmod)  # fmod::fmod also works

Copying DLLs (Windows)

FMOD is a SHARED library. After building or installing, copy the required DLLs next to your application.

add_custom_command(
  TARGET myapp POST_BUILD
  COMMAND ${CMAKE_COMMAND} -E copy_if_different
          ${fmod_SHARED_LIB}
          $<TARGET_FILE_DIR:myapp>
)

If the DLLs are missing, the program may fail to start or produce runtime errors.

Built‑in Variables

VariableDescription
fmod_INCLUDE_DIRSInclude directories for FMOD headers.
fmod_LIBRARIESImported library target(s) for linking.
fmod_SHARED_LIBPath to the runtime DLL to deploy with your app (Windows).
